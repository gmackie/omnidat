# Asterisk Config Scaffold

This directory is a planning scaffold, not a production-ready PBX config.

Files:

- `extensions-omnidat.conf`: Exchange 88 dialplan sketch.
- `pjsip-omnidat-lab.conf`: local lab SIP trunk/endpoint sketch.

Before use, adapt contexts, transports, codecs, endpoint names, and security
policy to the actual PBX host.
