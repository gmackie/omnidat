# OMNIDAT Identity

## Core Mark

```text
OMNIDAT
a gmacko corporation
```

OMNIDAT should feel like an 1980s financial data subsidiary: beige, formal,
over-integrated, confident, and faintly ominous. It is not a startup, hacker
club, or parody bank. It is a business-services corporation that already has a
file on you.


## Voice

OMNIDAT copy should be bureaucratic and operational. It should imply confidence
without explaining the joke.

Use:

- "account"
- "settlement"
- "terminal"
- "clearing"
- "records"
- "forms"
- "authorized access"
- "operator assistance"
- "exchange"
- "service desk"

Avoid:

- "cyber"
- "hacker"
- "AI"
- "web3"
- "startup"
- "fun"
- "retro experience"
- "immersive activation"

## Visual Direction

The in-world materials should look like they were printed by a conservative
enterprise vendor between 1982 and 1991:

- Monospace labels, dot matrix forms, or early laser-printer typography.
- Black, white, gray, green-screen amber, red warning stamps, and muted blue.
- Tables, account cards, accession numbers, line numbers, and operator initials.
- Rubber-stamp language: RECEIVED, AUTHORIZED, PENDING, HOLD, CLEARED.
- Signage that looks official enough to obey.
